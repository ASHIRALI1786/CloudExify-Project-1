The objective of this project is to analyze a retail sales dataset using Python and perform Exploratory Data Analysis (EDA) to generate business insights.

- Jupyter Notebook

- Matplotlib

## Dataset

- Cleaned missing values
- Removed duplicate records
- Converted date columns
- Performed statistical analysis
- Identified top-selling products
- Analyzed sales by category
- Analyzed sales by region
- Created visualizations
- Generated business insights

- Top 10 Products by Sales

- Identified the highest selling products.
- Compared sales performance across categories.
- Evaluated regional sales distribution.
- Observed monthly sales trends.
- Analyzed profit generated by different categories.
